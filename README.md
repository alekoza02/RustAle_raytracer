Progetto : Raytracer in Rust

Supporta:

- Geometrie:
  - Sfere
 
- Hardware:
  - CPU : multithread
 
- Materiali:
  - Diffuse
  - Metal
  - Glass
  - Light source

- Camera:
  - Movable
  - Antialiasing

- Post processing:
  - None
 
- Algoritmi:
  - Monte carlo's sample generazione
  - Riflettanza del vetro usando la legge di Brew

- Modalità di renderizzazione:
  - Standard completa
